Added five ⌘K quick-search entries for the Combat tab's reference material — Cover, Travel Pace,
Mounted Combat, Underwater Combat, and the Combat playbook. While a character sheet is open, search
any of them (in English or Italian) and it jumps to the Combat tab, opens the matching reference
section, and scrolls it into view — so a rule you need mid-play is one search away.
