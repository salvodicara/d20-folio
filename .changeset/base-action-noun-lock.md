Two Italian rules texts that still named an action by its old verb form are corrected to the official
2024 nouns: Haste's granted action now reads "l'azione Attacco … o Utilizzo" (was "Attaccare … o
Utilizzare"), and the Berserker Axe's rage compulsion says "l'azione Attacco". A build-time guard
now pins every base action's Italian name to the official IT SRD 5.2.1 noun and sweeps the Italian
rules text for the retired verb forms, so this class of wording drift can't return.
