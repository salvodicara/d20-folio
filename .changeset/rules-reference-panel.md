Added a "Rules reference" panel to the Play tab: an at-the-table quick reference for the situational
rules a player looks up mid-session — Cover, Mounted Combat, Underwater Combat, and Travel Pace.
These 2024 SRD facts have no per-character value to compute, so the app now simply shows them (in
English and Italian, both themes), with travel-pace distances converted to metric for Italian. This
also surfaces the Cover reference for the first time — it had shipped as data with nowhere to see it.
